🔥 Настройка Firebase

Для мультиплеера нужен Firebase Realtime Database. Это бесплатно!

1Иди на console.firebase.google.com → Создай проект
2Добавь веб-приложение (значок </>) → Скопируй firebaseConfig
3В проекте: Realtime Database → Создать → Начать в тестовом режиме
4Вставь свой firebaseConfig ниже и нажми Сохранить
⚠️ Без databaseURL мультиплеер не заработает!
